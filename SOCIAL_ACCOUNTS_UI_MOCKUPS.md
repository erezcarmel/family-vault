## Color Palette

### Brand Colors (from detectSocialNetwork)
- Facebook: `#1877F2` (bright blue)
- Twitter/X: `#000000` (black)
- Instagram: `#E4405F` (pink/red gradient)
- LinkedIn: `#0A66C2` (professional blue)
- YouTube: `#FF0000` (red)
- TikTok: `#000000` (black)
- Snapchat: `#FFFC00` (yellow)
- Pinterest: `#E60023` (red)
- Reddit: `#FF4500` (orange-red)
- WhatsApp: `#25D366` (green)
- Telegram: `#0088cc` (light blue)
- GitHub: `#181717` (dark gray)
- Discord: `#5865F2` (blurple)
- Twitch: `#9146FF` (purple)

### UI Colors (from Tailwind CSS)
- Success (green): `bg-green-50` border `border-green-200`
- Warning (red): `bg-red-50` border `border-red-200`
- Caution (yellow): `bg-yellow-50` border `border-yellow-200`
- Primary button: `bg-indigo-600` hover `bg-indigo-700`
- Secondary button: `bg-gray-200` hover `bg-gray-300`

## Typography

- Card title (network name): `text-lg font-semibold text-gray-900`
- Card subtitle: `text-sm text-gray-600`
- Card details: `text-xs text-gray-500`
- Form labels: `text-sm font-medium text-gray-700`
- Help text: `text-xs text-gray-500`
- Warning text: `text-sm text-red-800` / `text-yellow-800`
- Success text: `text-sm text-green-800`

## Icons

All icons from FontAwesome:
- Social network logos: Brand icons (free-brands-svg-icons)
- Edit: `faEdit` (regular)
- Delete: `faTrash` (regular)
- Close: `faTimes` (regular)
- Check: `faCheck` (solid)
- Warning: `faExclamationTriangle` (solid)
- Eye: `faEye` / `faEyeSlash` (solid)
- Plus: `faPlus` (solid)

## Spacing

- Card padding: `p-6` (1.5rem)
- Form field spacing: `space-y-6` (1.5rem between fields)
- Button spacing: `space-x-3` (0.75rem between buttons)
- Icon size in cards: `text-2xl` (1.5rem)
- Modal max width: `max-w-2xl` (42rem)

## Interaction States

### Buttons
- Default: Base color with shadow
- Hover: Darker shade, increased shadow
- Active: Pressed effect
- Disabled: Reduced opacity (0.5)

### Form Fields
- Default: Light gray border
- Focus: Indigo border with ring
- Error: Red border
- Disabled: Gray background

### Cards
- Default: White with shadow
- Hover: Increased shadow (`hover:shadow-lg`)

## Animations

- Smooth transitions on all hover states: `transition-all`
- Modal fade in/out
- Icon color transitions
- Shadow transitions on cards

## Accessibility

- All form fields have visible labels
- Required fields marked with asterisk (*)
- Help text provides context
- Warning messages have appropriate ARIA roles
- Keyboard navigation supported
- Focus indicators visible
- Color contrast meets WCAG AA standards
